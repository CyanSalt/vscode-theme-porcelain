## Usage

```jsonc
// .vscode/setting.json
{
  "window.autoDetectColorScheme": true,
  "workbench.preferredLightColorTheme": "Porcelain Light",
  "workbench.preferredDarkColorTheme": "Porcelain Dark",
}
```
