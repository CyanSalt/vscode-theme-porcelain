# vscode-theme-porcelain

Inspired by [Vitesse Theme for VS Code](https://github.com/antfu/vscode-theme-vitesse) and [Nuxt UI](https://ui.nuxt.com/).

## Usage

```jsonc
// .vscode/setting.json
{
  "window.autoDetectColorScheme": true,
  "workbench.preferredLightColorTheme": "Porcelain Light",
  "workbench.preferredDarkColorTheme": "Porcelain Dark",
}
```
